PetscWrap.get_range — Method
get_range(vec::Vec)

Wrapper to VecGetOwnershipRange

However, the result (rstart, rend) is such that vec[rstart:rend] are the rows handled by the local processor. This is different from the default PETSc.VecGetOwnershipRange result where the indexing starts at zero and where rend-1 is last row handled by the local processor.

PetscWrap.set_shell_mul! — Method

Wrapper for MatShellSetOperation for MATOP_MULT operation. f! signature must be f!(y::Vec, x::Vec) and must perform mul!(y, A, x).

Warning : to prevent the garbage collector from freeing the @cfunction, the latter is returned and you must store it in a variable:

c_func = set_shell_mul!(mat, f!)
